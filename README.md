Pysony
===============
python sony camera api

Install
====
using pip:

    pip install pysony

using repo:

    git clone https://github.com/Bloodevil/sony_camera_api.git
    python setup.py install

Usage
====

 simple example:

    >>> api = pysony.SonyAPI()
    >>> api.getAvailableApiList()
